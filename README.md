# Mission Control (ROS 2 Humble)

A ROS 2 Humble project demonstrating the core communication mechanisms in ROS 2.

## Features

- Topics (Publisher & Subscriber)
- Services (Emergency Stop)
- Actions (Autonomous Mission)
- Launch Files
- ROS 2 CLI Testing

## Platform

- Ubuntu 22.04 LTS
- ROS 2 Humble Hawksbill
- Python (rclpy)
